# Experiments

This directory contains earlier prototypes that informed the current app. They are
preserved for history and are **not** maintained.

## Contents

- **`sassy-argubot/`** — initial Chainlit-based prototype of the argument bot.
- **`chainlit-root/`** — root-level Chainlit configuration and translations from
  an early UI experiment.

The canonical, supported app lives at the repository root (`src/`, `backend/`).
